Overview

Purpose

This site is to easily help people find the cards they are looking for in an easier and more reliable way. This site will help people find up-to-date cards and the specific ones they are looking for, which will in turn help them find those cards on other sites where they can buy them.

Audience

This site is for people ages 8-40 or anyone interested in playing or collecting the trading card game Yu-Gi-Oh.

Dynamic elements

I'll be making this site mostly around a search engine that will allow users to search for cards by name and use a filter that can find cards by type, subtype, and/or attribute. This will be done using a JavaScript function that will filter through a list of cards and display the results on the page. I'll be using an OR logic for the search but make it so that it shows accurate results on top first. Then when clicked it will store it to display it onto another subpage where it will display all the cards you selected on there.
